# Computación II - 2026

Repositorio personal para la materia **Computación II** de la Universidad de Mendoza.

En este repositorio se encuentran los trabajos prácticos, ejercicios y actividades desarrolladas durante la cursada, con el objetivo de consolidar conocimientos en programación, herramientas de desarrollo y trabajo en entornos reales.

## Contenido

El repositorio incluye distintos trabajos prácticos y bloques de ejercicios enfocados en:

- Programación en Python
- Uso de la terminal y herramientas de desarrollo
- Manejo de argumentos de línea de comandos (`sys.argv` y `argparse`)
- Resolución de problemas prácticos mediante scripts
- Organización de proyectos con Git y GitHub

## Objetivo

El objetivo principal es desarrollar habilidades prácticas en programación y en el uso de herramientas profesionales, aprendiendo a construir programas claros, robustos y bien estructurados.

## Criterios de desarrollo

Todos los programas incluidos en este repositorio buscan:

- Ser claros y fáciles de entender
- Manejar errores de forma adecuada
- Tener una interfaz de uso simple desde la terminal
- Seguir buenas prácticas básicas de programación
